## Required store capsule assets to create before Steam store review

According to Steamworks graphical asset documentation, store page construction needs capsule and graphical assets such as:

```text
Header Capsule: 920 x 430
Small Capsule: 462 x 174
Main Capsule: 1232 x 706
Vertical Capsule: 748 x 896
```

Steam capsule artwork should have accurate dimensions and a readable product logo/name.
