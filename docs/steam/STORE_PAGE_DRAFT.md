# Steam Store Page Draft — Topoboardgame

This is a draft. Do not overclaim unfinished features. Steam review expects store text and screenshots to match the current build.

## Short description draft

Topoboardgame is a playable strategy laboratory where classic board games meet unusual spaces, boundaries, lattices, and systems. Play Chess, Go, Reversi, and experimental labs across new geometries with local robots, analysis tools, and online room support.

## Long description draft

Topoboardgame reimagines familiar board games on unfamiliar worlds.

Play classic and experimental variants of Chess, Go, and Reversi across spaces with different boundaries, lattices, and topological structures. Explore how strategy changes when movement, locality, symmetry, edge behavior, and connectivity are no longer fixed.

## Core features

- 2D and 3D Chess, Go, and Reversi variants.
- Multiple board geometries, boundaries, and lattice structures.
- Local play and local robot opponents.
- Position analysis tools for supported modes.
- Strategy & Systems Labs for playable experimental systems.
- Optional online rooms with guest or Google login.
- Local research tools for self-play, statistics, and trainable robot models.
