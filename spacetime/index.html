2+1D / 3+1D Space-Time Games

Original Chess, Go, Reversi, and Jump boards with a time layer attached.

These are not separate simplified prototypes. Each Play button opens the existing 2D or 3D game page, keeping its original board renderer, pieces, topology, boundaries, lattice controls, Local / Online / Robot modes, and rules. The +1D controls add Time schedule, optional Age, Go-only Time period, and optional noise where the game allows it.

For players: Time becomes a strategic resource. Pieces can schedule actions, optionally age, or in Go-only Time period mode appear and disappear on a fixed period.

For researchers: Spin, phase, parity, momentum, and Hamiltonian-style observables are kept in Strategy & Systems Labs, not in ordinary 2+1D / 3+1D games.

Noise

Noise is available only for +1D Go and Reversi. Chess and Jump keep deterministic movement.

How it works

Choose a card below. The opened game page will show a +1D Time Layer panel inside its normal controls. The page URL carries these settings so online rooms and shared links use compatible time rules.
